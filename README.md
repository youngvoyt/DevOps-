## Бизнес-задача

Предсказание дефолта заемщиков банка.  
Цель аналитики — построить модель, которая оценивает вероятность дефолта клиента на основе его анкетных и финансовых характеристик.

## Используемый датасет

В проекте используется датасет `course_project_test.csv` (финансовые данные заемщиков и целевой признак дефолта).  
Файл данных хранится локально и не коммитится в репозиторий (игнорируется через `.gitignore`).

## Структура проекта

- `data/` — файлы данных (сырые и обработанные), в Git не хранятся.
- `src/` — исходный код (скрипты загрузки и обработки данных).
- `notebooks/` — Jupyter-ноутбуки для исследований.
- `docs/` — документация и скриншоты выполнения лабораторной.

## Цели лабораторной работы

- инициализация репозитория и оформление README;
- создание структуры аналитического проекта;
- работа с ветками (`main`, `dev`, `feature/...`);
- моделирование конфликтов и их разрешение;
- сравнение версий файла описания данных `data/data_dictionary.md` с помощью `git diff`.
