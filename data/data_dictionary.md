# Data Dictionary – course_project_test.csv

## Общая информация

- **Источник данных**: локальный файл `course_project_test.csv`
- **Тематика**: кредитование, предсказание дефолта заемщиков.
- **Единица наблюдения**: один заемщик или одна кредитная заявка.

## Примерная структура полей

> В этом разделе зафиксированы типы и смысл полей для задачи предсказания дефолта.  
> При необходимости ты можешь скорректировать имена и описания под свой конкретный CSV.

| Имя поля          | Тип        | Описание                                                | Пример значения      |
|-------------------|-----------|---------------------------------------------------------|----------------------|
| client_id         | int       | Уникальный идентификатор клиента                       | 100001               |
| application_id    | int       | Уникальный идентификатор кредитной заявки              | 500123               |
| target_default    | int (0/1) | Целевой признак: 1 — дефолт, 0 — нет дефолта           | 0                    |
| gender            | string    | Пол клиента                                            | F                    |
| age_years         | int       | Возраст клиента в годах                                | 35                   |
| income_total      | float     | Ежемесячный или годовой доход клиента                  | 75000.0              |
| credit_amount     | float     | Сумма выданного кредита                                | 300000.0             |
| credit_term_months| int       | Срок кредита в месяцах                                 | 36                   |
| employment_years  | float     | Стаж работы клиента (в годах)                          | 5.5                  |
| marital_status    | string    | Семейное положение                                     | Married              |
| education_level   | string    | Уровень образования                                    | Higher education     |
|住房_status        | string    | Статус жилья (собственное, ипотека, аренда и т.п.)     | Own                  |

В дальнейшем этот словарь данных будет дорабатываться и расширяться.  
Разные версии файла `data_dictionary.md` будут сравниваться с помощью команды `git diff`.
